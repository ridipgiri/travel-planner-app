# AI Travel Planner

A simple travel planning application built using Gradio.

## Features

- Enter travel destination
- Select travel dates
- Choose budget
- Select travel interests
- Generate a suggested itinerary

## Project Structure

travel_planner/
|
|- requirements.txt
|- main.py
|- streamlit.py
+- README.md

## Installation

Install dependencies:

pip install -r requirements.txt

## Run the Application

python streamlit.py

## Output

The application launches a Gradio interface where users can enter travel details and receive a generated travel itinerary.

## Technologies Used

- Python
- Gradio
